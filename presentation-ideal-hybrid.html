us-east-1 (N. Virginia) — BACKUP / FAILBACK ONLY
Producer S3 Buckets (replicated from us-west-2) — Receives reverse CRR replication from west2. Purpose: backup, failback, compliance. No active platform services running.
Key Advantage: Iceberg metadata absolute path problem is eliminated entirely. Producers write to west2 S3 — metadata contains native west2 paths. No MRAP, no path substitution, no failover-time re-registration needed.
